Setting Up Langauge#

Via Assistant Bot#

  • Send /start to your assistant bot.
  • Click Languages Button.
  • Select your desired Language.
  • Done!

Via Command#

  • Check Language code in table above.
  • Do .setdb language lang_code to set your language.
  • Now restart your bot.
  • Done
